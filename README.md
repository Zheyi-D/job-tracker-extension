# 求职投递追踪

![MIT](https://img.shields.io/badge/license-MIT-green)

投完简历点一下，自动记录公司、岗位、链接和时间，同步到你的飞书多维表格，方便随时追踪投递进度。

## 安装

1. 下载本项目 [ZIP](https://github.com/Zheyi-D/job-tracker-extension/archive/refs/heads/main.zip) 并解压
2. Chrome 打开 `chrome://extensions`，右上角打开"开发者模式"
3. 点"加载已解压的扩展程序"，选择解压后的文件夹

## 首次配置（约 10 分钟，只需一次）

> 扩展装好后右键图标 → 选项，设置页里也有这份指南。

1. 打开 [飞书开放平台](https://open.feishu.cn)，进入开发者后台，**创建企业自建应用**（名称随意，如"求职追踪"）。创建后在"凭证与基础信息"页复制 **App ID** 和 **App Secret**。
2. 左侧"权限管理"，搜索并开通 `bitable:app`。
3. 左侧"版本管理与发布"，**创建版本并发布**（自己是管理员就自己审批通过）。⚠ 权限发布后才生效，这一步最容易忘。
4. 在飞书"我的空间"里新建一个**多维表格**（⚠ 不要建在知识库），按下面的字段建列：

   | 列名 | 类型 |
   |---|---|
   | 公司 | 文本 |
   | 岗位 | 文本 |
   | 投递时间 | 日期（勾选"显示时间"） |
   | 链接 | 超链接 |
   | 状态 | 单选：已投递 / 测评 / 笔试 / 一面 / 二面 / 三面 / HR面 / Offer / 已挂 / 已拒绝 |
   | 备注 | 文本（选填） |

   > 建议给"状态"列建一个看板视图，跟踪面试进度会很直观。

5. 打开这张表 → 右上角 **⋯ → 更多 → 添加文档应用** → 搜索你的应用 → 添加并设为**可编辑**。⚠ 漏掉这步后面会报错。
6. 把表格的浏览器地址复制下来，粘贴到扩展设置页的 app_token 输入框（会自动识别解析）。
7. 设置页填好四项凭证，点**测试连接**，看到"✓ 连接成功"就完成了。

## 使用

1. 在招聘官网投完简历
2. 点工具栏图标打开侧边栏，会自动抓取当前页面的公司、岗位、链接
3. 核对一下（橙色边框表示推测值，最好确认一下），点**保存到飞书**
4. 切换到下一个岗位页面会自动带入新信息，继续记录就行

## 其他

- 同一链接投过会有重复提示，但不会拦截
- 网络问题导致保存失败时可以先存本地，之后在设置页重试同步
- 设置页可以查看、导出、清空投递历史，也可以一键恢复默认状态
