📖 首次配置指南（点击展开）
  1. 打开 飞书开放平台 → 开发者后台 → 创建企业自建应用；在"凭证与基础信息"页复制 App ID 和 App Secret。
  2. 应用的"权限管理"页搜索并开通 bitable:app（查看、编辑和管理多维表格）。
  3. 创建版本并发布（自己是管理员则自己审批）。⚠ 权限在发布后才生效，这一步最容易漏。
  4. 在飞书"我的空间"新建多维表格（不要放知识库），按下方"字段要求"建列。
  5. 打开这张表 → 右上角 ⋯ → 更多 → 添加文档应用 → 搜索你的应用 → 添加并设为可编辑。⚠ 漏掉这步会报"无权访问"。
  6. 复制表格的浏览器地址，粘贴到下方 app_token 输入框——会自动解析出 app_token 和 table_id。
  7. 点击"测试连接"，看到"✓ 连接成功"即完成。

🔒 安全说明：App Secret 仅保存在本机浏览器的扩展存储中（不云同步、网页无法读取）。建议应用只开通 bitable 权限、只授权这一张表，将影响范围限制到最小。
